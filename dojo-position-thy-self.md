# Position they self

## 🧠 Concept
In Linux, every process has a current working directory (CWD).  
You can change this directory using the cd (change directory) command.

The symbol ~ represents your home directory, and the prompt shows your current location.

---

## 🎯 Goal
Navigate to a specific directory, then execute the challenge program from there.

---

## 🛠️ Steps

1. Change directory to the required path:
bash cd /challenge 

2. Run the program:
bash ./run 

---

## 💡 Key Takeaways
- cd is used to move between directories  
- Every process has a current working directory  
- You must be in the correct directory to run some programs  
- ./ is used to execute a file in the current directory  

---

## 🚩 Result
Successfully executed the program from the correct directory and captured the flag.
