# Intro to Commands

## Challenge:
Learn how to run basic commands in Linux.

## What I did:
- Opened the terminal in the dojo environment  
- Typed the command `hello`  
- Pressed Enter to execute the command  
- The system returned the flag after running the command  

## What I learned:
- How to execute commands in the Linux terminal  
- Commands are case-sensitive (e.g., `hello` ≠ `HELLO`)  
- After running a command, the shell returns to the prompt and waits for the next input
