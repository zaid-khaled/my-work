# Command History

## Challenge:
Learn how to use command history in the Linux terminal.

## What I did:
- Opened the terminal in the dojo environment  
- Used the up arrow key to access previous commands  
- Browsed through the command history  
- Found the injected flag inside the history  
- Copied and submitted the flag successfully  

## What I learned:
- The Linux terminal saves a history of all executed commands  
- You can navigate through past commands using the up/down arrow keys  
- Command history saves time by avoiding retyping long commands  
- History can sometimes contain useful information like flags in challenges
