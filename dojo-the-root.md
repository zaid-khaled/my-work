# The Root - pwn.college

**Description:**  
Run the `pwn` binary located in the root directory `/` using an absolute path to get the flag.

**Solution:**  
The binary is in `/`, so I executed it using:
```bash
/pwn
```

**Result:**  
The program ran successfully and returned the flag.

**What I Learned:**  
- Absolute paths in Linux  
- Executing binaries from root directory
