# Intro to Arguments

## Challenge:
Learn how to use command arguments in Linux.

## What I did:
- Opened the terminal in the dojo environment  
- Ran the command `hello hackers`  
- The command used `hello` as the main command and `hackers` as an argument  
- After executing the command, I received the flag  

## What I learned:
- Commands in Linux can take arguments (extra inputs)  
- The first word is the command, and the rest are arguments  
- Example: `echo Hello` → `echo` is the command, `Hello` is the argument  
- Commands can accept multiple arguments  
- Understanding arguments is important for using commands effectively
