Challenge: Restarting the Challenge  
Platform: pwn.college  

Goal:  
Understand how to restart a challenge after breaking the environment and use the correct information to complete it.

What I did:  
- Started the challenge and opened the terminal in the Desktop environment  
- Entered a password (it was incorrect)  
- The system revealed the correct password but removed the flag and broke the environment  
- Copied the correct password to the clipboard  
- Restarted the challenge to reset the environment  
- Opened the terminal again  
- Pasted the correct password from the clipboard  
- Successfully completed the challenge  

What I learned:  
- Some challenges are designed to fail on the first attempt  
- Restarting the environment is an important skill in cybersecurity labs  
- How to use clipboard data effectively after restarting  
- Mistakes are part of the learning process in hacking environments
